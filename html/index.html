Let's Start with kubernetes with serverless and command less functionality.
Yeah you read it right. No need to remember any command , no need to maintain any k8s node, Just Start...

What?

Kubernetes Friendly is a website for all kubernetes related facility. It is complete serverless website.

Why?

Here you can deploy your services with k8s facility, without knowing any k8s command and even you don't need to worry about the backend client and server node of k8s.

How?

We have command line as well as Graphical support for all the k8s services. Also we have provided you various Documentation, K8s chatsheets, and many Blogs, So Start learning and Implementing!!
